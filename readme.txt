Предварительно необходимо для данной папки поставить node.js, а также убедиться,что поставлены все компоненты, находящиеся в server.js в разделе const ... require(...) и что установлена база данных sqlite3.
Всё это ставится через npm(включая базу), который тоже нужно поставить.
Nodemon (тоже поставить) нужен для автоматического перезапуска сервера.
И каждый раз когда меняется файл myBase.sql нужно в cmd ввести команду:
sqlite3.exe myBase.db < myBase.sql
Запуск сервера идёт с cmd: дойдя до папки, в которой хранится файл server.js нужно вызвать команду nodemon server.js
